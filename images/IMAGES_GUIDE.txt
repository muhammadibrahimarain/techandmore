========================================
IMAGE OPTIMIZATION TIPS
========================================

Before uploading:
1. Resize to 800x600 pixels (exact)
2. Compress using TinyPNG.com (reduce file size)
3. Aim for under 200KB file size
4. Save as JPG (smaller) or PNG (better quality)

This ensures fast page loading!
